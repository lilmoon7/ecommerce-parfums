# Site E-commerce de Parfums

Projet de développement d'un site e-commerce responsive, réalisé dans le cadre de ma formation.

## Objectif

L'objectif était d'intégrer une maquette (inspirée de divers designs) en HTML5, CSS3 et JavaScript, en mettant l'accent sur une interface utilisateur propre et fonctionnelle.

## Fonctionnalités

* Page d'accueil et catalogue produits
* Fiches produits détaillées
* Gestion du panier d'achat (ajout, suppression, mise à jour des quantités)
* Site 100% responsive (adapté aux mobiles et tablettes)

## Technologies utilisées
* HTML5
* CSS3 (Flexbox, Grid)
* JavaScript (ES6+) pour la logique du panier
